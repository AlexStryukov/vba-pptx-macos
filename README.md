# vba-pptx-macos
Cтандартные команды создания формул  в MacOS сломаны. Они вызывают "Ошибку 5" или теряют фокус окна. Этот репозиторий предлагает решение: вместо того чтобы создавать формулу с нуля (что вызывает сбои), макрос использует метод "Клонирования шаблона". Он копирует скрытую формулу с первого слайда, вставляет её как новый объект и меняет текст внутри.
